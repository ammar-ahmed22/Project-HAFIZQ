# Project HAFIZQ
Kijiji/Craigslist style website created as part of exercise for Dreamschools interview. 

## Tech Stack
#### Front-End
-React
-Bootstrap (styling)
-Boxicons (icons)

#### Back-End
-Node.js
-Express.js (server)
-Mongoose (database structuring)

#### Database
-MongoDB

#### Hosting
-Google Firebase (front-end)
-Heroku (back-end/server)
-MongoDB Atlas (cloud database)
